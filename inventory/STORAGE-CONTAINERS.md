## Labeling System

Each container gets a label following the naming convention from CATALOGING.md:

```
[ZONE]-[CATEGORY]-[NUMBER]
Example: GARAGE-SHELF-WEST-WORKSHOP-01
```

Labels should include:
- Zone label (from 3D model)
- Primary category
- Sequential number
- Tier level (T1/T2/T3/T4)
